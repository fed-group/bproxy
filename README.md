# bproxy

## 集抓包、调试、mock 于一身的前端开发利器

## 开发

```bash
npm i pnpm -g
# nrm use taobao 使用淘宝源安装依赖
pnpm install

# 1、先构建一下 birdge 和 utils 包
yarn build
# 2、启动服务端
yarn start-server
# 3、启动 Web
yarn start-client
```
